sapho by sroberts

A homebrewed cyber threat intelligence solution built on DokuWiki to enable teams doing intelligence driven computer network defense.

Sapho was built as an off hours project to manage considerable intelligence developed from computer network defense. Building up on the considerable resources of DokuWiki Sapho automatically generates a framework of wiki resources for capturing and analyzing cyber threat intelligence and responding.

Dependencies

  • Python - Developed on Python 2.7
  • DokuWiki - Developed on 2011-05-25a "Rincewind"
  • DokuWikiXMLRPC - https://github.com/chimeric/dokuwikixmlrpc

Install

- Follow DokuWiki installation instructions found at http://www.dokuwiki.org/Install - Enable XMLRPC interface and an account - Run templater.py as "python templater.py -w -u -p --setup This should set up the start and initially linked pages.
